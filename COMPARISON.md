# Comparison

| Library                                              | Description                                                               |
| ---------------------------------------------------- | ------------------------------------------------------------------------- |
| [`chess.js`](https://www.npmjs.com/package/chess.js) | full chess engine — SAN parsing included with permissive and strict modes |
| [`chessops`](https://www.npmjs.com/package/chessops) | TypeScript library with `parseSan`/`makeSan` — part of larger library     |
| [`chess`](https://www.npmjs.com/package/chess)       | algebraic notation driven engine — SAN and UCI formats                    |
